Typography

Inter – Used for headers, menus, buttons, and all UI elements

Serif Family (Data & Charts)

Buttons

Interactive button components with hover states and accessibility considerations.

Usage Guidelines

  • Primary: Main call-to-action buttons (Submit, Save, Confirm)
  • Secondary: Supporting actions (Cancel, View More, Learn More)
  • Outline: Tertiary actions or when you need less visual weight
  • Disabled: Inactive state for buttons that cannot be interacted with

Form Inputs

Minimal, underline-style input fields with clear focus states.

Design Notes

  • Underline-only style reduces visual clutter
  • Red accent color on focus draws attention
  • Consistent padding and spacing for alignment

Navigation Components
